# Strict Symmetric BEAR Tape Gate — Backtest & Verdict

**Date:** 2026-07-13 · **Clause 0: RESEARCH ONLY — no live-code change proposed.**
**Dataset:** same 64-day Skylit archive / 537 final-system fires (331 bull /
206 bear) that validated the bull tape gate. Real UW option-dollar EV
(`pnl_atfire` / `entry_atfire·100`). Script: `bear_gate.py`.

**Question:** the bull gate blocks BULL fires when SPY+QQQ+SPXW are ALL below
prior close. Its docstring calls it "the symmetric completion of the G7-PC
bear gate: bears already require spot < prior close." Does a STRICT symmetric
BEAR gate — block BEAR fires when SPY+QQQ+SPXW are ALL *above* prior close
(puts into unanimously green tape) — clear the same bar AND add material
blocking beyond what G7-PC already blocks?

## VERDICT: NO — arithmetically a NO-OP. Do not arm. Do not add to watchlist.

The strict all-3-above bear gate blocks **exactly 0 of 206 bear fires** in the
64-day dataset. It is not "weak" or "underpowered" — it is a mathematical
subset of what G7-PC already blocks, so its incremental value is **provably
zero, not merely small.** Every relaxed variant that *would* block something
blocks **winners** (wrong sign).

## (B) The mirror cell does not exist

| bears, indexes above prior close | n | EV | win |
|---|---|---|---|
| 0/3 above (all-red tape) | 188 | −6.5% | 52% |
| 1/3 above | 13 | +6.6% | 69% |
| 2/3 above | 5 | +31.4% | 60% |
| **3/3 above (strict gate target)** | **0** | **—** | **—** |

There are **zero** bear fires with all three indexes above prior close. The
strict gate has nothing to block. `system −5.2% → kept −5.2%` (unchanged).

Two things to note, both fatal to the idea:
1. **The mirror cell is empty**, so it cannot show the bull cell's −15%
   negative-EV signature — there is no cell.
2. **The bear tape gradient runs the OPPOSITE way to the bull thesis.** For
   bulls, more-red tape = worse (0/3-above is the −15% poison cell). For bears,
   more-*red* tape is the worst cell (0/3-above = −6.5%), and the *greener* the
   tape, the *better* bears did (1/3 +6.6%, 2/3 +31.4%). "Green tape is bad for
   puts" is not just redundant here — it is the wrong sign.

## (C) Incremental over G7-PC: provably zero (the decisive result)

G7-PC already requires the **fired ticker's own** spot < its prior close. Every
fire in this system IS one of SPY / QQQ / SPXW. Therefore any bear that G7-PC
*allows* has at least one index (its own) below prior close ⇒ `n_above ≤ 2`
**by construction**. All-3-above is impossible for a G7-PC-allowed bear.

Cross-tab of all 206 bears (`fired_ticker_above` × `indexes_above`):

```
indexes_above →     0     1     2     3
fired_above 0     188    13     5     0      ← all 206 bears live here
fired_above 1       0     0     0     0      ← empty: G7-PC blocks these
```

- **G7-PC ALLOWS** all 206 bears (fired ticker below its own prior close).
- Bears the strict all-3 gate would block that G7-PC also allows: **n = 0.**
- The strict gate's blocking set ⊆ complement of G7-PC's, and the overlap the
  gate would *add* is the empty set. **Incremental blocking = 0 fires, exactly.**

This is why the bull gate is not redundant but the bear mirror is: bulls had
**no** per-ticker tape requirement (docstring: "bulls had no tape requirement"),
so the bull gate contributed 117 fresh blocks. The bear side already carries
G7-PC, which strictly dominates the proposed all-3 mirror.

## (D) Stability cuts — not applicable (n=0)

Odd/even, H1/H2, and per-ticker (SPY/QQQ/SPXW) all have **0 blocked fires** in
the strict bear cell, so there is nothing to test for stability. The bull gate
cleared 4/4 stability + placebo because it had a populated, robust −15% cell;
the bear mirror has no cell at all.

## Relaxed variants also fail (wrong sign)

If one tried to salvage a bear tape gate by loosening the threshold:

| relaxed bear "green tape" gate | blocks n | block-set EV |
|---|---|---|
| block bears with ≥2 indexes above | 5 | **+31.4%** |
| block bears with ≥1 index above | 18 | **+8.3%** |

Both block **positive-EV** bears — a relaxed mirror would *destroy* edge. There
is no threshold at which a green-tape bear gate helps.

## Bottom line

- **Same bar as the bull gate?** No. The bull gate had a populated −15% cell
  (n=117) that passed stability + placebo. The bear mirror cell is empty.
- **Material incremental blocking beyond G7-PC?** No — provably zero fires, by
  construction (fired ticker ∈ {SPY,QQQ,SPXW} guarantees its own index is below
  when G7-PC allows the bear).
- **Action:** do not arm; do not add to the forward-validation watchlist. The
  docstring line "symmetric completion of the G7-PC bear gate" is a
  *conceptual* symmetry, not an actionable one — G7-PC already IS the bear-side
  tape requirement, and it strictly dominates the all-3 mirror. The asymmetry
  in the codebase (per-ticker gate for bears, per-basket gate for bulls) is
  correct and complete as-is.
